最长子序列匹配（524）

· 2022-01-01 · # 双指针 # leetcode刷题记录

题目

双指针解法

思路：每次从数组中取出来一个字符串与目标字符串对比，sl指向目标字符串，tl指向取出来的子字符串，若匹配sl和tl同时加1，若不匹配++sl。经过该过程后判断tl是否和子字符串长度一样，一样则说明匹配，然后更新结果即可。

class Solution {
public:
    string findLongestWord(string s, vector<string>& dictionary) {
        string result;
        int maxNum = 0;
        for(int i = 0;i < dictionary.size();i++){
            string temp = dictionary[i];
            int tL = 0;
            int sL = 0;
            while(sL < s.length() && tL < temp.length()){
                if(s[sL] == temp[tL]){
                    ++sL;
                    ++tL;
                }else{
                    ++sL;
                }
            }
            if(tL == temp.length() && tL >= maxNum){
                if(tL > maxNum){
                    maxNum = tL;
                    result = temp;
                }else{
                    if(temp < result){
                        result = temp;
                    }
                }
               
            }
        }
        return result;
        
    }
};
